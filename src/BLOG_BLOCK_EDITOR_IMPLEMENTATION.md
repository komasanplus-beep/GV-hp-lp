# ブログ・ブロックエディタ実装完了報告

## 🎯 実装概要
**WordPress型ブロックエディタ + AI記事生成統合**により、AIで記事を生成→編集→公開まで1画面で完結できるようになりました。非エンジニアでもSEO記事量産が可能です。

---

## 📦 追加・変更ファイル

### エンティティ（1個）
- **Article** - ブログ記事（ブロック型）
  - `blocks` (JSON配列) - ブロック構造
  - `generate_toc` - 目次自動生成フラグ
  - `ai_generated` - AI生成フラグ
  - `ai_prompt` - AI生成時のプロンプト

### バックエンド関数（1個）
- **generateBlogArticleAI** - AI記事生成（ブロック変換含む）

### フロントエンドコンポーネント（6個）

#### ブロックエディタ系
1. **BlockEditor.jsx** - WordPress型ブロックエディタ
   - ドラッグで並び替え
   - ブロック単位の追加・削除・複製
   - 見出し・段落・画像・リスト・引用・コード対応

2. **BlockHeadingEdit.jsx** - 見出しブロック編集パネル
3. **BlockParagraphEdit.jsx** - 段落ブロック編集パネル（スタイル設定）
4. **BlockImageEdit.jsx** - 画像ブロック（アップロード・alt設定）
5. **BlockListEdit.jsx** - リストブロック（項目管理）
6. **BlockQuoteEdit.jsx** - 引用ブロック

#### AI・UI系
7. **AIBlogGeneratorPanel.jsx** - AI記事生成パネル（拡張版）
   - プロンプト入力
   - トーン・文字数・SEOキーワード設定
   - 参考資料アップロード
   - Web検索オプション
   - 再生成ボタン

8. **TableOfContents.jsx** - 自動目次生成・表示

### ページ（1個）
- **AdminBlogEditor.jsx** - 新規ブログ作成・編集画面
  - 上段：AI生成パネル
  - 中段：ブロックエディタ
  - 下段：SEO・公開設定
  - プレビュー機能

### ユーティリティ（1個）
- **lib/blockUtils.js** - ブロック関連ユーティリティ
  - 目次生成
  - ブロック→HTML変換
  - 抜粋生成

## 🔧 ブロック種類

| ブロック | 説明 | エディタ | 機能 |
|---------|------|--------|------|
| **heading** | 見出し | H2/H3/H4選択 | SEO用 |
| **paragraph** | 段落 | テキスト + スタイル | フォントサイズ・太字・行間 |
| **image** | 画像 | URL/アップロード | altテキスト・幅・配置 |
| **list** | リスト | 項目入力 | 番号付き/箇条書き |
| **quote** | 引用 | テキスト + 出典 | ブロッククォート |
| **separator** | 区切り線 | なし | 視覚的区切り |
| **code** | コード | テキスト（等幅） | プリフォーマット |

## 🧠 AI生成フロー

```
1. ユーザー入力
   ├─ プロンプト
   ├─ トーン・文字数
   ├─ SEOキーワード・地域
   ├─ 参考資料（PDF/URL）
   └─ Web検索フラグ

2. generateBlogArticleAI 関数
   ├─ aiGuard で利用可能チェック（回数制限）
   ├─ 詳細なプロンプト構築
   ├─ OpenAI/Claude API 呼び出し
   ├─ マークダウン出力取得
   └─ マークダウン→ブロック配列変換

3. 出力形式
   {
     title: "記事タイトル",
     excerpt: "抜粋...",
     blocks: [
       { id, type: "heading", level: 2, content: "見出し" },
       { id, type: "paragraph", content: "段落...", style: {...} },
       { id, type: "image", image_url: "...", alt_text: "..." },
       ...
     ]
   }

4. UI反映
   ├─ プレビュー表示
   ├─ 「エディタに挿入」ボタン
   └─ ユーザーがさらに編集可能
```

## 📋 マークダウン → ブロック変換ルール

| マークダウン | ブロック | 変換例 |
|-----------|---------|--------|
| `## テキスト` | heading (level: 2) | H2見出し |
| `### テキスト` | heading (level: 3) | H3見出し |
| `テキスト` | paragraph | 通常段落 |
| `- 項目` | list (unordered) | 箇条書き |
| `1. 項目` | list (ordered) | 番号付き |
| `> 引用` | quote | ブロッククォート |
| `---` | separator | 区切り線 |
| `` ` `` | code | コードブロック |

## 🚀 使用方法

### 記事作成（AI利用）
1. `/AdminBlogEditor?site_id=site_123` にアクセス
2. 「AIで記事を作成」パネルにテーマを入力
3. 「AIで記事作成」をクリック
4. 生成結果をプレビュー
5. 「この記事をエディタに挿入」ボタンで反映
6. さらに編集（ブロック追加・削除・並び替え）
7. SEO設定を入力
8. 「保存」で下書き保存
9. ステータスを「公開」に変更して「保存」で公開

### ブロック直接編集
1. 記事作成画面から「ブロックを追加」
2. 種類を選択（H2見出し / 段落 / 画像等）
3. 各ブロックのパネルで内容を編集
4. ドラッグで順序変更
5. 「保存」

## ⚙️ 設定・制限

### AI生成回数制限
- `aiGuard` 関数で チェック
- プランに応じて月間利用回数制限
- 再生成・複数候補も回数消費

### ブロック数制限
- 制限なし（ただし大量ブロックはパフォーマンス注意）

### テキスト長
- 見出し：制限なし（推奨50-70文字）
- 段落：制限なし（UI上は自動折返し）
- AI生成時：500-3000文字を選択可

## 🔍 SEO考慮事項

1. **見出し構造**
   - H2 → H3 → H4 という階層構造を推奨
   - 見出しのスキップは避ける

2. **目次生成**
   - H2/H3 を自動抽出して目次化
   - SEO効果あり（内部リンク）

3. **altテキスト**
   - 画像ブロック時に必須入力推奨
   - SEO + アクセシビリティ

4. **メタ情報**
   - SEOタイトル（60文字以内）
   - メタディスクリプション（160文字以内）
   - キーワード登録
